# Forschungs‑ und Verteidigungsposter

## Zweck
Dieser Ordner enthält alle wissenschaftlichen Poster, die im Rahmen der Zeitfeld‑Dissertation erstellt werden.  
Die Poster dienen zur Präsentation der zentralen Modelle, Ergebnisse und Visualisierungen auf:

- Konferenzen  
- Workshops  
- Verteidigungen  
- internen Forschungsmeetings  

## Typische Inhalte
Ein Poster umfasst üblicherweise:

- Überblick über das Zeitfeldmodell  
- Darstellung der fünf Zeitfeld‑Schichten  
- Visualisierung galaktischer Zeitgradienten  
- Solare und planetare Zyklusstrukturen  
- Interferenzdiagramme  
- Tensorfeld‑Darstellungen  
- Zusammenfassung der wichtigsten Ergebnisse  

## Format
Poster können in folgenden Formaten vorliegen:

- `.pdf` (Standard)  
- `.png` (Export)  
- `.svg` (Vektorgrafiken)  
- `.pptx` (Poster‑Vorlagen)

## Verwendung
Poster aus diesem Ordner werden genutzt für:

- wissenschaftliche Präsentationen  
- Konferenzbeiträge  
- Verteidigungsunterlagen  
- grafische Zusammenfassungen für Publikationen  

## Status
Dieser Ordner dient als Sammelstelle für alle Poster‑Versionen und zugehörige Grafiken.
